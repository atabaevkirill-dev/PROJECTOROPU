# 🎯 Контроллер Управления Прожектором

Десктопное приложение для управления поворотным устройством (ОПУ) и релейной платой по TCP-сети. Построено на **CustomTkinter** с тёмной темой интерфейса.

---

## ✨ Возможности

### Поворотное устройство (ОПУ TL.0250)
- 🕹️ **8-направленный D-pad** — кардинальные и диагональные направления, непрерывное движение при удержании
- ⏹️ **Кнопка STOP** — мгновенная остановка
- 🎚️ **Регулировка скорости** — слайдер 5–100 °/сек
- 📍 **Отображение позиции** — текущие углы Pan/Tilt в реальном времени
- ⚙️ **Настройки ОПУ:**
  - Позиционирование (переход по углу)
  - Режим качания (swing)
  - Ограничения скорости и углов
  - Статус устройства (температура, напряжение, состояние)
  - Самодиагностика
  - Пресеты Pelco-D

### Релейная плата (RelayX3)
- 🔘 **До 3 каналов** — кнопки ВКЛ/ВЫКЛ с отображением реального состояния
- 🔄 **Опрос состояния** — синхронизация с аппаратным состоянием каждые 2 секунды
- ⚙️ **Настройки реле:**
  - Видимость каналов
  - Пользовательские названия каналов
  - Таймеры на канал (Ч:М:С + циклы 1–999/∞)
  - Параллельный / последовательный режим выполнения
  - Кнопки «Запустить все» / «Остановить все»

### Общее
- 💾 **Сохранение настроек** (JSON) — имена, видимость, скорость, таймеры сохраняются при выходе
- 🔌 **Автопереподключение** к обоим устройствам
- 🎨 **Современный тёмный UI** — карточные панели, акцентные линии, невидимая прокрутка в настройках

## 📋 Требования

- **Python 3.10+**
- **customtkinter >= 5.2.0**

---

## 📦 Установка

```bash
pip install -r requirements.txt
```

---

## 🚀 Запуск

```bash
python control_app.py
```

---

## ⚙️ Конфигурация

- Файл **`settings.json`** создаётся автоматически при первом запуске и хранит все пользовательские настройки.
- IP-адреса и порты устройств заданы как константы в начале файла `control_app.py`:

| Устройство | IP | Порт |
|---|---|---|
| ОПУ TL.0250 | 192.168.1.115 | 9762 |
| RelayX3 | 192.168.1.114 | 9761 |

---

## 📡 Протоколы

- **ОПУ TL.0250** — ASCII-протокол сервисного управления (поворот, позиционирование, статус, диагностика).
- **RelayX3** — бинарный протокол **Pelco-D** (7-байтовые пакеты, контрольная сумма без учёта байта 0xFF, адрес по умолчанию = 1). Управление через TCP с RS-485 фреймингом.
